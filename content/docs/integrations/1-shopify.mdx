## Step 1: Access Shopify Admin and Create API Credentials

To integrate Tiankii with Shopify, follow these steps to create an app and generate your API credentials.

### Access Shopify Admin

1. Go to [Shopify Admin](https://admin.shopify.com).
2. Navigate to **Settings** â†’ **Apps and sales channels**.
3. Click on **Develop apps**.

![Access Shopify Admin](/assets/image.png)

---

### Create the App

1. Click **Create an App**.
2. Enter the name: **`tiankii-app`**.
3. Save the app.

![Create App](/assets/image2.png)

---

### Configure the App

1. Open the newly created app.
2. Go to the **Configuration** tab.
3. Under **Admin API integration**, enable the following permissions:
   - **read_orders**
   - **write_orders**
4. Click **Save** to confirm.

![Configure App](/assets/image3.png)
