## Change Invoice Payment Method

### Authorization

**API Key**  
This request is using the API Key from the collection **TIANKII API**.

### Endpoint

**PATCH** `/v1/invoice/pm`  
`{{HOST}}/v1/invoice/pm`

### Request Body

Send the request body in raw JSON format. Below is an example:

```json
{
  "invoiceId": "{{invoiceId}}",
  "paymentMethodId": "{{newPaymentMethodId}}"
}
```

### Request Example

```javascript
var axios = require('axios');
var data = '{\r\n  "invoiceId": "AQNCTzam1RuXzoR9LtwgUg",\r\n  "paymentMethodId": "BTC_StrikeLike"\r\n}';

var config = {
  method: 'patch',
maxBodyLength: Infinity,
  url: 'https://api.md.tiankii.com/v1/invoice/pm',
  headers: { },
  data : data
};

axios(config)
.then(function (response) {
  console.log(JSON.stringify(response.data));
})
.catch(function (error) {
  console.log(error);
});


```
