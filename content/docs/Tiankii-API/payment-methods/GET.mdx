## Payment Methods

### Authorization

**API Key**  
This folder is using the API Key from the collection **TIANKII API**.

### Endpoint

**GET** `/v1/:storeId/payment-method/actives`  
`{{HOST}}/v1/:storeId/payment-method/actives`

### Authorization

**API Key**  
This request is using the API Key from the collection **TIANKII API**.

### Path Variables

| Variable  | Description            |
|-----------|------------------------|
| `storeId` | Unique identifier for the store. |

### Request Example

```javascript
var axios = require('axios');

var config = {
  method: 'get',
maxBodyLength: Infinity,
  url: '{{HOST}}/v1/:storeId/payment-method/actives',
  headers: { }
};

axios(config)
.then(function (response) {
  console.log(JSON.stringify(response.data));
})
.catch(function (error) {
  console.log(error);
});

```
