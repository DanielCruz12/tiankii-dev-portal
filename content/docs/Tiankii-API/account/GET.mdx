## Get Account Information

### Authorization

**API Key**  
This folder is using the API Key from the collection **TIANKII API**.

### Endpoint

**GET** `/me`
`{{HOST}}/v1/store/me`

### Authorization

**API Key**  
This request is using the API Key from the collection **TIANKII API**.

```javascript
var axios = require("axios");

var config = {
  method: "get",
  maxBodyLength: Infinity,
  url: "{{HOST}}/v1/store/me",
  headers: {},
};

axios(config)
  .then(function (response) {
    console.log(JSON.stringify(response.data));
  })
  .catch(function (error) {
    console.log(error);
  });
```
